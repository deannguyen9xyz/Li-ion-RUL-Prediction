# Li-ion-RUL-Prediction
Analysis of Lithium-ion battery degradation using NASA datasets. Features data visualization, statistical curve fitting, and Machine Learning models to predict Remaining Useful Life (RUL).

....updating....

Script 1: 01_visualization.py => Visualize the raw data to understand physical degradation.

Script 2: 02_correlation.py => Mathematically prove which physical parameters drive the aging process.

Script 3: 03_curve_fitting.py => Use fitting curve to estimate total life based on the capacity trend.

Script 4: 04_machine_learning.py => Train an ML model to predict RUL for an unseen battery.
